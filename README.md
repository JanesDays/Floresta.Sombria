Floresta Sombria — Projeto Front-End

Floresta Sombria é um projeto front-end interativo, desenvolvido com HTML, CSS e JavaScript puro, que apresenta lendas do folclore brasileiro por meio de um carrossel animado, com foco em design, organização de código e experiência visual.
O projeto foi criado com o objetivo de demonstrar habilidades em desenvolvimento web, manipulação do DOM e estruturação de interfaces visuais.

 Demonstração do Projeto

- Site estático com navegação por carrossel
- Conteúdo visual e textual integrado
- Interface focada em storytelling cultural

-Demonstrar domínio de front-end básico e intermediário
-Aplicar JavaScript sem frameworks
